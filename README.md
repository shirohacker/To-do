Vue3 CLI를 이용하여 제작한 To-do List 입니다.

백엔드는 별도로 구성하지 않고 로컬 json server를 이용하였습니다.

본 프로젝트는 계속 개발중에 있습니다.

# Function

- Todo 추가
- Todo 삭제
- Todo 검색

# Project setup

```
npm install
```

# Compiles and hot-reloads for development

```
npm run serve -- --port 3000
```
